ATM Interface

This project is a console-based ATM Interface developed using Java.

Features:
- User login (ID & PIN)
- Check transaction history
- Withdraw money
- Deposit money
- Transfer money
- Exit option

Technologies Used:
- Java
- OOP Concepts
- BlueJ IDE

This project was completed as part of the Oasis Infobyte Java Development Internship.
